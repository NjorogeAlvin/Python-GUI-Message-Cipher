# Message-Cipher
A simple desktop app for encrypting and decrypting message using a secret key

## Features
- Encrypts and Decrypts using a Fernet key
- Generate a new key with the click of a button
- Re-use of previous keys

## Requirements
- Python3
- customtkinter
- cryptography

## Installation
pip install customtkinter cryptography

## Usage
python main.py

1. Generate a key or paste an already existing one
2. Type message and hit encrypt
3. Share message and the key to decrypt it
